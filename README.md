# Helix Website

Landing page project for Helix, built with Next.js, React, shadcn/ui, and Tailwind CSS.

## Quick Start

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Notes

- Update content in `components/sections/*` to match your brand voice.
- Global site settings live in `config/site.ts`.
